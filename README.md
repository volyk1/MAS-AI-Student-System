# MAS AI Student System

Система підтримки прийняття рішень в e-commerce.

## Опис

MAS AI Student System — це проєкт на базі ШІ-агентів для:

- сегментації клієнтів і підготовки пояснювальних AI-підказок;
- тестування студентів із подальшим аналізом відповідей;
- формування персоналізованих запитів для навчальної підтримки.

Проєкт використовує модульну структуру в папці src, де окремі скрипти відповідають за обробку даних, оцінювання, генерацію підказок та запуск локального інтерфейсу.

## Інструкція з інсталяції

1. Клонувати репозиторій:

```bash
git clone <your-repo-url>
cd "MAS AI Student System"
```

2. Створити віртуальне середовище:

```bash
python -m venv .venv
```

3. Активувати віртуальне середовище:

```bash
# Windows (PowerShell)
.\.venv\Scripts\Activate.ps1

# Windows (Git Bash)
source .venv/Scripts/activate
```

4. Встановити залежності:

```bash
pip install -r requirements.txt
```

## Інструкція з запуску

1. Запуск базового сценарію:

```bash
python src/main.py
```

2. Запуск локального інтерфейсу:

```bash
python src/ui_copy_prompt.py --port 8000
```

3. Відкриття інтерфейсу:

- Відкрийте index.html через локальний сервер за адресою http://localhost:8000

## Налаштування API-ключів

Перед запуском скопіюйте файл .env.example у .env і заповніть значення ключів:

```bash
cp .env.example .env
```

Після цього відредагуйте .env та вкажіть реальні API-ключі (наприклад OPENAI_API_KEY).

Важливо: файл .env не повинен потрапляти в Git (він вже доданий у .gitignore).
